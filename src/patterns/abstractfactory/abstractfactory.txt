Шаблон Abstract Factory (Абстрактная фабрика) - это паттерн проектирования,
который позволяет создавать семейства взаимосвязанных объектов, не раскрывая их конкретных классов.

Он относится к категории порождающих шаблонов и предоставляет интерфейс для создания семейств взаимосвязанных объектов
без указания их конкретных классов.

Абстрактная фабрика позволяет создавать объекты, которые взаимодействуют друг с другом так,
чтобы они были совместимы между собой.


В примере абстрактные продукты - это стул и диван, представленные интерфейсами Chair и Sofa.

У нас есть две конкретные реализации этих продуктов для двух стилей мебели: "Модерн" и "Викторианский".

Абстрактная фабрика FurnitureFactory объявляет методы для создания стула и дивана.

Конкретные фабрики ModernFurnitureFactory и VictorianFurnitureFactory реализуют эти методы, создавая соответствующие продукты.

Затем в main методе мы создаем фабрики и используем их для создания и использования мебели в разных стилях.
